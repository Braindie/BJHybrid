### NSURLProtocol

### JavaScriptCore与UIWebView
OC与JS中的方法可以相互调用，不过JS中要有特定的对象作为OC的代理对象

### WebViewJavascriptBridge
只能使用UIWebView<br/>
通过给JS中注入特点代码，然后匹配方法名可以实现JS与OC的相互调用，并且每次调用都会有回调
